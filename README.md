# PopX React App

A pixel-perfect ReactJS implementation of the PopX mobile-style interface.  
This project replicates the Adobe XD mockups with clean, modular code and seamless navigation.

---

## 🚀 Features
- Mobile app interface centered on desktop
- Pixel-perfect design with responsive layout
- Seamless navigation using React Router
- Modular structure with separate pages (`Welcome`, `SignIn`, `SignUp`, `AccountSettings`)
- Ready for deployment on Vercel
